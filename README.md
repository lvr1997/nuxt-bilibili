# Nuxt 3 仿B站的移动端项目

技术栈：Nuxt3 + Vant

## 构建工具

Make sure to install the dependencies:

```bash

# pnpm
pnpm install

```

## 项目开发

Start the development server on `http://localhost:3000`:

```bash
# pnpm
pnpm run dev

```
## 项目总结

- [x] Nuxt环境搭建
- [x] Nuxt是用来做什么的
- [x] Nuxt的路由系统
- [x] ts的基本语法

## 打包上线

Build the application for production:

```bash

# pnpm
pnpm run build

```

生产环境预览

```bash

# pnpm
pnpm run preview

```
